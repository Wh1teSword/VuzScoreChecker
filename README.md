# VuzScoreChecker
МОШ проект
Данный проект - это интерактивная веб-платформа, которая помогает абитуриентам оценить шансы поступления в вузы на основе суммарного балла ЕГЭ и данных прошлых лет.
Пользователь вводит свой балл и выбирает город, после чего система:
подбирает подходящие вузы и программы,
сравнивает балл с проходными значениями,
показывает вероятность поступления,
определяет место в конкурсном рейтинге,
оценивает, проходит ли абитуриент по количеству бюджетных мест
